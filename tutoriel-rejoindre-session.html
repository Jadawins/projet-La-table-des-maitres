🚪 Rejoindre une session

Trouvez une partie ouverte au recrutement, rejoignez-la (publique ou avec mot de passe) et associez votre personnage pour être prêt à jouer.

~3 min de lecture · 7 étapes
7 étapes
1

Ouvrir "Mes Sessions (Joueur)"

Dans le menu de gauche, cliquez sur "Mes Sessions (Joueur)". Vous arrivez sur la page qui liste les sessions disponibles et celles que vous avez déjà rejointes.

2

Parcourir les sessions disponibles

Chaque carte de session affiche : le nom de la partie, le MJ, le nombre de joueurs actuels et le statut (Recrutement ouvert, En cours, Terminée).

3

Filtrer par statut

Utilisez les onglets en haut de la liste pour n'afficher que les sessions "En recrutement" — ce sont les seules que vous pouvez rejoindre. Les sessions "En cours" sont visibles mais ne peuvent plus être rejointes.

4

Voir les détails d'une session

Cliquez sur une carte pour afficher la page de détail : description de la campagne, joueurs inscrits, règles de la partie. Vous pouvez décider de rejoindre depuis cette page.

5

Session publique — Rejoindre directement

Sur une session publique, cliquez simplement sur "Rejoindre". Vous êtes ajouté à la liste des joueurs immédiatement. Le MJ peut vous voir dans son interface.

6

Session privée — Entrer le mot de passe

Si la session est 🔒 Privée, un champ mot de passe apparaît. Demandez le code au MJ qui organise la partie. Entrez-le et cliquez "Rejoindre".

7

Associer son personnage

Après avoir rejoint, choisissez quel personnage vous jouez dans cette session via le menu déroulant. Vous pouvez changer votre personnage tant que la session n'a pas démarré.

⚠️
Attention
Un même personnage ne peut pas être utilisé dans deux sessions simultanées. Si votre personnage est déjà engagé dans une partie active, il n'apparaîtra pas dans la liste de sélection.
ℹ️
Gestion côté MJ
En tant que MJ, créez une session depuis le Dashboard MJ. Vous pouvez inviter les joueurs par le code de session, gérer les inscriptions et démarrer la partie quand tout le monde est prêt.
