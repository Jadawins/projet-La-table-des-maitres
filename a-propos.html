🧙 À propos

La Table du Maître est une application web conçue par et pour les passionnés de jeux de rôle sur table. L'objectif est simple : centraliser tous les outils dont un Maître du Jeu et ses joueurs ont besoin, dans une interface moderne, accessible et gratuite.

Le projet est né de la frustration de jongler entre des dizaines d'onglets, de PDF et de feuilles de personnage papier lors des sessions. La Table du Maître réunit tout au même endroit.

👤 Qui sommes-nous ?

Ce projet est développé par Wiwi, développeur passionné de JDR, en dehors de tout cadre commercial. La Table du Maître est un projet indépendant, développé sur le temps libre, avec passion et beaucoup de café.

Le code source s'appuie sur des technologies modernes et ouvertes : HTML/CSS/JS côté frontend, Node.js / Express côté API, Supabase pour l'authentification et MongoDB pour les données de jeu.
