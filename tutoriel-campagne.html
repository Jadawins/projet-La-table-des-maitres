📋 Créer une campagne

Le Tableau de bord MJ est votre hub central. Créez une campagne avec l'éditeur de blocs, liez vos sessions, préparez vos rencontres et suivez la progression de vos joueurs.

~5 min de lecture · 7 étapes
7 étapes
1

Accéder au Tableau de bord MJ

Dans le menu, cliquez sur Tableau de bord MJ. Ce hub central regroupe toutes vos campagnes, vos sessions actives, les statistiques de vos joueurs et votre journal de campagne.

2

Créer une nouvelle campagne

Dans la colonne de navigation gauche, cliquez sur + Nouvelle campagne. Une fenêtre s'ouvre pour saisir le nom et la description de votre campagne. Confirmez — la campagne apparaît dans votre liste et l'éditeur de blocs s'ouvre automatiquement.

3

L'éditeur de blocs

Le cœur de la campagne est son éditeur de blocs. Chaque bloc est un élément de contenu que vous pouvez réorganiser librement. Cliquez sur un type de bloc dans la barre d'outils pour l'ajouter :

⚔️ Rencontre — description du combat, monstres, récompenses
🎲 Jet demandé — type de compétence, DC, conséquences
💰 Loot — trésors, objets magiques, monnaie
👤 PNJ — personnage non-joueur avec traits et motivations
🗺️ Lieu — description d'un endroit avec ambiance
📝 Texte libre — notes, lore, description narrative

💡
Auto-save
Vos modifications sont sauvegardées automatiquement toutes les 30 secondes. Vous pouvez aussi forcer la sauvegarde avec le bouton 💾 Enregistrer dans la barre d'outils.
4

Créer et lier une session

Dans la colonne de navigation, cliquez sur + Nouvelle session. Renseignez le nom, la date prévue, et choisissez si elle est publique ou privée (avec mot de passe optionnel). La session sera liée à votre campagne active.

5

Gérer les joueurs et le dashboard de session

Sélectionnez une session active dans la liste pour voir le dashboard de session : cartes de chaque joueur avec leurs PV actuels, conditions, niveaux, et actions rapides (envoyer un message, infliger des dégâts, envoyer un flash).

6

Le journal de campagne

Chaque action importante (combat terminé, XP distribués, joueur rejoint…) est automatiquement enregistrée dans le journal. Vous pouvez aussi ajouter des entrées manuelles pour noter des événements narratifs ou des décisions de vos joueurs.

7

Distribuer XP et gérer la progression

Depuis le panneau contextuel droit, utilisez le module Distribution XP pour attribuer des points d'expérience à l'issue d'une rencontre. Choisissez entre XP classique ou système au jalon (montée de niveau automatique sans XP). Les fiches joueurs se mettent à jour instantanément.

ℹ️
Lancer un combat
Pour démarrer un combat depuis une session active, accédez à l'Écran MJ (bouton dans le dashboard de session). C'est depuis cet écran que vous gérez l'initiative et le tracker en temps réel. Voir le tutoriel Gérer un combat (MJ).
